# Лендинг турфирмы «счастливого_Путешествия»

Адаптивный одностраничный сайт для туристического агентства.

## Особенности

- **Адаптивность**: мобильная версия (до 768px) и десктоп
- **7 секций**: хедер, главный экран, туры, преимущества, условия, форма, футер
- **SCSS-архитектура**: переменные, миксины, разделение на файлы
- **Миксины**: переиспользуемые стили для кнопок и контейнера

## Запуск

1. Склонируйте репозиторий
2. Откройте `index.html` в браузере

Для компиляции SCSS (если нужны изменения):
```bash
sass scss/styles.scss css/styles.css --watch
